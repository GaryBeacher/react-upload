# react-upload

图片上传组件

### Installation
``` sh
npm i input-moment --save
```

**Notice:** This module requires [moment](https://www.npmjs.com/package/moment) as a [peerDependency](https://docs.npmjs.com/files/package.json#peerdependencies).

### Demo

``` sh
npm run start
```

http://localhost:8888/example/

### Usage
``` javascript
<Upload />
```
